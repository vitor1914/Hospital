# Programa de Hospital em C#

## Introdução

Este é um programa de hospital desenvolvido em C# que permite cadastrar enfermeiras com autenticação e usuários com informações pessoais. Além disso, o programa gera automaticamente um arquivo PDF para o Ministério da Saúde se o usuário atender a certos critérios.

## Requisitos

- Visual Studio ou IDE C# semelhante instalada.
- Bibliotecas de terceiros, como iTextSharp para a geração de PDFs.

## Funcionalidades

1. Cadastro de Enfermeira:
   - Autenticação de enfermeira com login e senha.
   - Armazenamento seguro das credenciais.

2. Cadastro de Usuário:
   - Nome completo.
   - Idade.
   - CPF.
   - Identidade.

3. Geração de Relatório para o Ministério da Saúde:
   - Se o usuário tiver 65 anos ou apresentar comorbidades, um relatório em PDF será gerado automaticamente e enviado ao Ministério da Saúde.
   - O relatório inclui informações sobre o usuário e seu estado de saúde.

## Como Usar

1. Clone o repositório para sua máquina local.

2. Abra o projeto no Visual Studio ou na IDE C# de sua preferência.

3. Compile e execute o programa.

4. Use a interface do programa para cadastrar enfermeiras e usuários.

5. O programa irá automaticamente gerar um PDF para o Ministério da Saúde quando aplicável.
